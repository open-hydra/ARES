A finite-volume CFD solver written in modern Fortran

Compressible Flow Solver

Euler and Navier-Stokes equations on multi-block structured grids, with steady-state and time-accurate computation modes.

High-Order Numerics

MUSCL reconstruction with six flux limiters. Six Riemann solvers including preconditioned and rotated HLLC variants.

Real-Fluid Thermodynamics

Tabulated real-fluid equation of state via the FLINT library: properties evaluated from pressure-enthalpy lookup tables.

Turbulence Modelling

RANS: Spalart-Allmaras (with R/RC/QCR/rough variants), Menter SST, Wilcox k-ω 2006, and the SSG-LRR Reynolds-stress model.

Low-Mach Preconditioning

Weiss-Smith-type preconditioning recovers accuracy and convergence at low Mach number, where standard density-based schemes stall.

Parallel & Multigrid

Shared-memory OpenMP and distributed-memory MPI, with geometric multigrid acceleration for fast convergence to steady state.

Up and running in minutes

ARES ships with a single install script that handles all dependencies, compiles the Fortran core, and produces the solver binary.

Installation guide → Quick start
terminal 
# Clone and build
$ git clone https://github.com/open-hydra/ARES.git
$ cd ARES && ./install.sh build --compilers=gnu

# Run a simulation
$ ./bin/ARES

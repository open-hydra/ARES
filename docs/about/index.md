# About

ARES (**A** **R**eal-fluid **E**quations **S**olver) is an open-source compressible-flow solver developed within the **Hydra** CFD ecosystem.

## Citing ARES

If you use ARES in academic work, please cite the repository (see `CITATION.cff`) and acknowledge the underlying [FLINT](https://github.com/MarcoGrossi92/FLINT) and [ORION](https://github.com/MarcoGrossi92/ORION) libraries.
